# holy-js-childs

Это репа с микрофронтами child-apps

## Установка зависимостей

Перед началом работы установите зависимостью с помощью `yarn install`

## Работа с проектом

- `yarn start` - запустить локальный сервер
- `yarn build` - сделать продакшен сборку

## Project structure

* `child-apps/holy-js-childs` - исходный код микрофронтов
  * `index.ts` - входная точка микрофронта
  * `postcss.js` - postcss конфиг микрофронта
